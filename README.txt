--- SANGSATIONAL SIMULATOR ---

Nom de projet : Sangsational simulator

Contributeurs : Clara Bomy & Mathilde Christiaens

Mise en service : Pour lancer le projet, il suffit de l'importer sur votre éditeur et de cliquer sur le bouton run.
		  Vous trouverez alors un menu vous permettant de choisir quel type de simulation vous souhaitez lancer.
    		  Ensuite, il vous suffira d'appuyer sur la touche ENTRER pour lancer la simulation. 
			Bon amusement !

PS :  A l'issue d'une simulation, un fichier de logs sera disponible à la racine du projet pour voir en détail les actions des personnages durant leur partie.
